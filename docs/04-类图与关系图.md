## 4. 运行态与持久态关系

```mermaid
flowchart TD
    ACTION[用户动作/冷启动恢复] --> BUSINESS[DownloadManager / InstallManager / UpgradeManager]
    BUSINESS --> STATE[StateCenter\n运行态单一真相]
    BUSINESS --> PERSIST[Repository / Store\n持久化真相]
    STATE --> APPMANAGER[AppManager]
    PERSIST --> APPMANAGER
    APPMANAGER --> UI[页面 ViewModel / Task Center]
```

当前需要特别区分：

- `StateCenter` 保存的是运行态快照
- `Repository`、`DownloadStore`、`InstallSessionStore`、`LocalStoreFacade` 保存的是可恢复的持久态
- 页面看到的数据多数由 `AppManager` 二次聚合后输出
