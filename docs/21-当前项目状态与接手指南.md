## 5. 关键架构决策

- **Repository**: `RealAppRepository` 代理到 3 个 DataSource，`FakeAppRepository` 仅用于测试
- **状态管理**: `DefaultStateCenter` + `StateReducer` 归约，自动推导 primaryAction 和 statusText
- **下载控制**: `DownloadExecutionControl` 支持 pause/cancel + `registerInterrupt` 连接级中断
- **数据持久化**: `VersionedJsonStore` 统一管理 JSON 文件的 schema 版本、迁移、并发锁、原子写
- **策略拦截**: `DefaultPolicyCenter` 检查 WiFi/驻车/存储，升级同时校验下载+安装策略
