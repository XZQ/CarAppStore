## 说明

- 文中图示使用 Mermaid 表达，可在支持 Mermaid 的 Markdown 工具中直接预览。
- 当前工程约束保持不变：
  - MVVM
  - 不使用 Hilt
  - 不使用 Navigation
  - 使用 FragmentManager
  - 使用 AppContainer 做手动依赖注入
- 当前业务层固定为 7 个核心模块：
  1. 下载模块
  2. 安装模块
  3. 升级模块
  4. 应用管理模块
  5. 状态中心
  6. 策略中心
  7. Repository
